# Centro Project Context

## Product
Centro is an AI-powered sales phone agent SaaS.

Companies route inbound sales calls to Centro.
Centro answers calls, qualifies leads, collects information, scores opportunities, summarizes calls, and helps sales teams follow up.

## Current Stack
- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Prisma ORM
- PostgreSQL
- Clerk Authentication
- Twilio
- Vercel deployment
